# 젠스파크(GenSpark) 디자인 프롬프트 — Upstage 공식 브랜드 가이드 기반

> Upstage Brand Guide (Airtable) 공식 데이터를 기반으로 작성되었습니다. `[주제]`, `[내용]` 플레이스홀더를 교체해서 사용하세요.

## 프롬프트 전문

```
다음 디자인 시스템을 정확히 따라서 프레젠테이션 슬라이드를 제작해주세요.
이 규칙은 Upstage AI의 공식 브랜드 가이드라인에서 직접 추출한 것입니다.

============================================================
1. 전체 브랜드 아이덴티티
============================================================

이 프레젠테이션은 B2B 엔터프라이즈 AI 기업(Upstage)의 제품/회사 소개 덱입니다.

핵심 디자인 철학: "Visually Relaxing" (시각적으로 편안한 디자인)
- 요소를 과도하게 사용해 많은 메시지를 한번에 전달하기보다는, 
  여백(White Space)을 활용하여 핵심 메시지에 집중할 수 있는 디자인을 추구합니다.
- 여백을 두어 눈이 쉴 수 있는 공간을 제공하며, 요소들의 중요도와 우선순위를 
  명확하게 구분합니다.
- 정돈된 레이아웃을 통해 사용자가 자연스럽게 글을 읽어나가도록 돕고,
  화면 상의 요소들이 일관된 느낌을 줄 수 있도록 합니다.
- 균형 잡힌 구성을 통해 명확한 시각적 흐름을 제공하고, 
  복잡한 정보를 안정적으로 제공합니다.

무드 키워드: Clean, Minimal, Professional, Trustworthy, Visually Relaxing

============================================================
2. 공식 컬러 시스템 (반드시 이 컬러만 사용)
============================================================

[브랜드 컬러 — 절대 변형 불가]
- Ups Purple: #805CFB

  → 절대로 변형(색조, 투명도 조절 등)해서는 안 됨.
  → 배경색으로 전면 사용하지 말고, 포인트/엑센트 컬러로만 사용.

[Primary Color — Jacksons Purple 스케일]

[컬러 사용 핵심 규칙]
- Purple(#805CFB)은 배경 전면 사용 금지 → 포인트 컬러로만
- 밝은 배경: Jacksons Purple 50~200 (#ECF0FF ~ #C2CBFF)
- 어두운 배경 필요 시: Secondary Gray 600~700
- 과도한 컬러 사용 자제 → 메인 메시지에 집중할 수 있도록
- 빨강, 초록, 노랑, 오렌지 등 원색 절대 금지

3. 공식 타이포그래피
============================================================

[브랜드 폰트]
- 브랜드/이벤트 타이틀 전용: ES Peak
  → 일관된 브랜딩을 위해 메인 타이틀에 ES Peak 사용
  → 다른 폰트로 대체 금지

[본문(Body) 폰트]
- 영문 위주: Geist (Google Fonts)
- 한영 혼합: Noto Sans KR
- 이 두 폰트를 우선 사용. 불가능할 경우 유사한 분위기의 산세리프 대체 가능.
  (예: Inter, Pretendard, Apple SD Gothic Neo)

[폰트 크기 가이드]
| 요소              | 크기        | Weight   | 색상                    |
|-----------------|-----------|----------|------------------------|
| 슬라이드 제목        | 36~44pt   | Bold     | Gray 600 (#1C2537)     |
| 섹션 구분 제목       | 44~56pt   | ExtraBold| Gray 700 (#0D1320)     |
| 부제/설명문         | 16~18pt   | Regular  | Gray 500 (#434C60)     |
| 본문 텍스트         | 14~16pt   | Regular  | Gray 500 (#434C60)     |
| 섹션 라벨(좌상단)     | 12~14pt   | Regular  | Ups Purple (#805CFB)   |
| KPI 숫자          | 48~64pt   | Bold     | JP 400 (#7578FF)       |
| KPI 설명          | 12~14pt   | Regular  | Gray 400 (#979CAE)     |
| 테이블 텍스트        | 14pt      | Regular  | Gray 500 (#434C60)     |
| 각주/출처          | 10~11pt   | Regular  | Gray 400 (#979CAE)     |
| 페이지 번호         | 10pt      | Regular  | Gray 400 (#979CAE)     |

[정렬 규칙]
- 모든 제목, 부제, 본문: 좌측 정렬이 기본
- 카드 내부 짧은 레이블/제품명만 가운데 정렬 허용
- 테이블 숫자: 가운데 정렬 / 테이블 텍스트: 좌측 정렬

============================================================
4. 레이아웃 시스템
============================================================

[기본 규격]
- 슬라이드: 16:9 와이드스크린
- 마진: 상하좌우 최소 0.6인치 (약 55px)
- 콘텐츠 좌측 시작점: 약 0.7인치에서 일관되게
- 요소 간 간격: 0.3~0.5인치 일관 유지
- 여백을 넉넉하게 — 빈 공간이 많은 것이 이 디자인의 특징

[White Space 원칙 — 공식 가이드]
- DO: 시각적 공백을 두어 유저가 한 번에 한 가지 메시지에 집중하도록 설계
- DON'T: 복잡한 레이아웃으로 많은 메시지를 한번에 전달하려는 시도

[Visual Stability 원칙 — 공식 가이드]
- 정돈된 레이아웃으로 자연스러운 읽기 흐름 제공
- 균형 잡힌 구성으로 복잡한 정보도 안정적으로 전달
- 화면 요소들의 일관된 느낌 유지

============================================================
5. 로고 사용 규칙 (공식 가이드)
============================================================

- 로고는 형태, 색상, 스타일을 절대 변형할 수 없음
- 다른 그래픽 요소와 결합하지 않음
- 밝은 배경(Light theme): 원본 퍼플 로고 또는 블랙 로고 사용
- 어두운 배경(Dark theme): 화이트 로고 사용
- 제품 로고(Solar LLM, Document Parse, Information Extract, AI Space)는 
  각 제품의 공식 로고를 사용

============================================================
6. 슬라이드 유형별 레이아웃 상세
============================================================

[유형 A] 커버 슬라이드 (표지)
- 배경: Jacksons Purple 50 (#ECF0FF) 단색
- 좌측 상단: Upstage 퍼플 로고 (작은 사이즈)
- 로고 아래: 메인 제목 (44~56pt, ES Peak 또는 Geist Bold, Gray 700)
- 제목 아래: 부제 (16~18pt, Geist Regular, Gray 500)
- 좌측 하단: 날짜 (14pt, Gray 400)
- 슬라이드 하단 약 25~30%: 추상적 그라데이션 블러 장식
  (Jacksons Purple 50~200 + 연한 아이보리 톤의 부드러운 블러 텍스처)
- 우측 하단 세로: 저작권 "© 2025 Upstage Co., Ltd." (10pt, 세로 회전)

[유형 B] 목차 슬라이드
- 배경: Jacksons Purple 50 (#ECF0FF) 전체
- 좌측 하단: "Contents" (72~80pt, ExtraBold, Gray 700)
- 우측 중앙~하단: 목차 항목 리스트
  - 대항목: Bold 16~18pt, 번호 포함
  - 소항목: Regular 14pt, 들여쓰기
  - 페이지 번호: 우측 정렬

[유형 C] 섹션 구분 슬라이드
- 배경: 그라데이션 블러 이미지 (JP 50~200 + 아이보리 추상 텍스처) 전체
- 중앙에서 약간 좌측: 큰 흰색 라운드 사각형 카드
  - 크기: 슬라이드의 약 65% 너비, 55% 높이
  - 모서리: 16~20px 라운드
  - 배경: #FFFFFF
  - 내부 좌측 하단: 섹션명 (44~52pt, ExtraBold, Gray 700)
- 장식 없음, 텍스트와 카드만으로 구성

[유형 D] 제품 소개 카드 슬라이드
- [유형 C]와 동일한 배경 (그라데이션 블러 + 흰색 카드)
- 카드 내부:
  - 상단: "Products" (16pt, Regular, Gray 500) + 가로 구분선 (Gray 200)
  - 중간: 제품 공식 아이콘 (40~50px)
  - 아이콘 아래: 제품명 (40~48pt, ExtraBold, Gray 700)
  - 제품명 아래: 한 줄 태그라인 (16pt, Regular, Gray 500)
  - 전체 좌측 정렬, 카드 좌측 하단 뭉침

[유형 E] 일반 콘텐츠 슬라이드 (가장 빈번)
- 배경: Jacksons Purple 50 (#ECF0FF)
- 좌측 상단: 섹션 라벨 (12~14pt, #805CFB Ups Purple)
  예: "Upstage", "Solar LLM", "Document Parse", "Information Extract", "AI Space"
- 라벨 아래: 슬라이드 제목 (36~40pt, Geist Bold, Gray 600 #1C2537)
- 제목 아래: 부제/설명 (14~16pt, Regular, Gray 500)
- 콘텐츠 영역:
  - 카드/박스 배경: #FFFFFF 또는 Gray 100 (#F8F7FD)
  - 보더: 1px Gray 200 (#E9E8F0) 또는 미세 박스 쉐도우
  - 모서리: 8~16px 라운드
- 우측 하단: 페이지 번호 (10pt, Gray 400)

[유형 F] KPI/숫자 강조 슬라이드
- [유형 E] 기본 구조
- 2x3 또는 3x2 KPI 박스 그리드
  - 박스 배경: JP 50 (#ECF0FF) 또는 Gray 100 (#F8F7FD)
  - 카테고리명: 14pt, Bold, Gray 600, 가운데 정렬
  - 핵심 숫자: 48~64pt, Bold, JP 400 (#7578FF)
  - 설명: 12pt, Regular, Gray 400, 가운데 정렬
- 모든 박스 동일 크기, 수평 정렬

[유형 G] 비교 테이블 슬라이드
- [유형 E] 기본 구조
- 테이블:
  - 헤더: JP 300 (#9CA7FF) 배경, 흰색 텍스트 Bold
  - 자사 열: JP 200 (#C2CBFF) 배경으로 하이라이트, 수치 Bold
  - 일반 행: #FFFFFF, 행 구분 1px Gray 200
  - 테이블 전체 라운드 8~12px
  - 각주: 10~11pt, Gray 400

[유형 H] 카드 그리드 슬라이드
- [유형 E] 기본 구조
- 카드: #FFFFFF 또는 Gray 100 배경
- 보더: 1px Gray 200, 라운드 12px
- 카드 내부: 제목 (18~22pt, Bold, 가운데) + 설명 (14pt, Regular, 가운데)
- 카드 간격: 16~24px, 모든 카드 동일 사이즈

[유형 I] 3열 유즈케이스/기능 비교 슬라이드
- [유형 E] 기본 구조
- 3개 열, 각 열:
  - 상단: 카테고리 (Bold 18pt) + 아이콘 (JP 400 또는 Gray 700)
  - 배경: JP 50 (#ECF0FF) 또는 Gray 100
  - 내부 미니 카드 3개: #FFFFFF, 1px 보더, 14pt 텍스트

[유형 J] 아키텍처/다이어그램 슬라이드
- [유형 E] 기본 구조
- 다이어그램 스타일:
  - 박스: 라운드 사각형, #FFFFFF, 1px Gray 200 보더
  - 화살표: 1~2px, Gray 400 또는 JP 400
  - 주요 노드: JP 100~200 (#DDE3FF ~ #C2CBFF) 배경
  - 텍스트: 12~14pt, 가운데 정렬

[유형 K] 로고 그리드 슬라이드
- [유형 E] 기본 구조
- 격자형 로고 배열 (6~7열)
- 각 셀: #FFFFFF, 라운드 8px, 간격 4~8px
- 로고 크기 일정, 중앙 정렬

[유형 L] 엔딩 슬라이드
- 배경: JP 50 (#ECF0FF)
- 좌측 상단: Upstage 로고
- 로고 아래: 마무리 메시지 (44~56pt, ExtraBold, Gray 700)
- 하단: 가로 구분선 (Gray 200) + 저작권 (10pt, Gray 400)

============================================================
7. 반복되는 디자인 패턴 (일관성 필수)
============================================================

- 섹션 라벨: 모든 콘텐츠 슬라이드 좌상단에 Ups Purple (#805CFB)으로 
  현재 섹션명 표시
- 카드/박스: 라운드 8~16px, #FFFFFF 또는 Gray 100, 미세 보더/쉐도우
- 테이블: 헤더 컬러 배경(JP 300), 자사 열 하이라이트(JP 200), 라운드
- 그라데이션 배경: 섹션 구분 + 제품 소개 카드에만 사용. 
  일반 콘텐츠에서는 JP 50 단색만.
- 아이콘: 라인 또는 미니멀 필 스타일. JP 400(#7578FF) 또는 Gray 700.
  별도 배경 원/사각형 없이 아이콘만 단독.
- 장식적 밑줄/언더라인: 제목 아래 절대 금지
- 그림자: 거의 사용하지 않거나 극히 미세하게

============================================================
8. 하지 말 것 (Anti-patterns)
============================================================

- ❌ Ups Purple(#805CFB)을 배경 전면에 사용
- ❌ 브랜드 컬러를 변형(색조, 투명도 변경)
- ❌ 로고의 형태, 색상, 스타일 변경 또는 다른 그래픽과 결합
- ❌ 그라데이션 배경을 일반 콘텐츠 슬라이드에 사용
- ❌ 제목 아래 장식적 밑줄/가로선
- ❌ 원색(빨강, 초록, 노랑, 오렌지) 사용
- ❌ 과도한 컬러 사용 (메인 메시지에 집중 방해)
- ❌ 모든 슬라이드에 동일 레이아웃 반복
- ❌ 이모지, 클립아트 스타일 일러스트
- ❌ 본문 가운데 정렬 (좌측 정렬 기본)
- ❌ 과도한 볼드 (제목, KPI, 강조에만)
- ❌ 텍스트로 슬라이드 가득 채움 (여백 충분히!)
- ❌ 불릿 포인트 남발 (카드, 그리드, 테이블 우선)
- ❌ 다크 모드/어두운 배경 전면 사용
- ❌ 복잡한 레이아웃 (한 번에 한 가지 메시지 원칙)
- ❌ ES Peak 이외의 폰트로 브랜드 타이틀 작성

9. 슬라이드 구성 순서 예시
============================================================

10. 커버 [유형 A]
```
